# Écosystème Alpha FLE — Formateurs de Français

Site statique gratuit de soutien pédagogique aux formateurs bénévoles de français, alphabétisation, FLE et littérisme.

## Philosophie

Le site a été conçu comme une mallette de terrain : des repères courts, des supports adaptables et une posture d'accompagnement fondée sur la dignité, la non-discrimination, la bienveillance et l'utilité sociale.

## Déploiement GitHub Pages

1. Créer un dépôt GitHub nommé `ecosysteme-alpha-fle`.
2. Copier tout le contenu de ce dossier à la racine du dépôt.
3. Vérifier `_config.yml` :
   - `baseurl: "/ecosysteme-alpha-fle"`
   - `url: "https://VOTRE-UTILISATEUR.github.io"`
4. Activer GitHub Pages dans `Settings > Pages`.
5. Le site sera disponible à l'adresse : `https://VOTRE-UTILISATEUR.github.io/ecosysteme-alpha-fle/`.
