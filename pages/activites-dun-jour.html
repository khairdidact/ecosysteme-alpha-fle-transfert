Activités d'un jour

S'AIDER DE L'IA

Un outil simple pour séquencer en quelques minutes une séance d'alphabétisation adaptée à votre public du jour — même si vous êtes débutant en pédagogie

Former au français dans un accueil de jour demande une flexibilité extrême : le public est volatil, les niveaux sont hétérogènes, et la progression pédagogique classique est souvent rompue par les aléas du quotidien.

Cette procédure vous permet de générer en quelques minutes un séquençage de séance complet et adapté grâce à l'intelligence artificielle Claude. Suivez simplement les 3 étapes ci-dessous.

Étape 1

Créer un compte Claude

Claude est l'assistant IA recommandé. Il est gratuit pour commencer — vous pouvez générer plusieurs séquences par jour sans abonnement.

🔐 Créer un compte Claude gratuit 🚀 Accéder à Claude
💡 Version gratuite ou payante ?
La version gratuite suffit pour 3 à 5 séquences par jour. La version Pro (20$/mois) est utile pour un usage quotidien intensif.
Étape 2

Télécharger la ressource source

Téléchargez l'unité MaClé ALPHA correspondant à votre séance (PDF). Ce fichier sera joint à Claude pour que l'IA s'appuie sur son contenu.

Nommez le fichier clairement : Unité_n01_MaClé_Alpha_A1.1.pdf

📘 Les ressources MaClé ALPHA sont disponibles gratuitement sur la page dédiée du site.
📥 Accéder à MaClé Alpha
Étape 3

Copier-coller le prompt

Dans Claude, joignez le PDF (icône trombone), puis copiez-collez le prompt ci-dessous en complétant les champs entre crochets.

À partir de la ressource fournie et des données suivantes :

Contexte de la séance
Date : [JJ/MM/AAAA]
Support source : [Ex: Unité n°01 Ma Clé Alpha A1.1]
Thématique : [Ex: LIRE, Écrire | Mes urgences]
Objectifs : [Ex: J'identifie mes besoins]
Effectif : [Ex: 8-12 personnes]
Niveaux : [Ex: infra-alpha + débutants]
Public : [Ex: Femmes seules avec enfants]
Contrainte matérielle : [Ex: Pas de vidéoprojecteur]

Agis en expert en alphabétisation. Propose un séquençage de 2h (collation incluse) : accueil (15'), découverte (25'), atelier pratique (40'), trace écrite (25'), collation (15'). Privilégie l'oral et l'utilitaire.
📎 Comment joindre le PDF ? Cliquez sur l'icône trombone dans Claude, sélectionnez votre fichier, puis collez le prompt.
