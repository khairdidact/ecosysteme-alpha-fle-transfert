# Reprise non destructive du site

Cette version repart du dépôt original `ecosysteme-alpha-fle` et conserve les fichiers, contenus, assets et liens pédagogiques existants.

Interventions réalisées :

- ajout d'une couche CSS d'harmonisation visuelle sans suppression de contenu ;
- amélioration du responsive : menu mobile, grilles adaptatives, boutons et cartes sur petits écrans ;
- amélioration du JavaScript du menu mobile : fermeture au clic extérieur, touche Escape, liens externes sécurisés ;
- correction du lien ScoLibris vers `https://www.scolibris.fr/` ;
- remplacement du lien absolu interne GitHub Pages vers les ressources MaClé Alpha par un chemin interne au site ;
- mise à jour de `_config.yml` et du `sitemap.xml` avec l'adresse réelle `https://khairdidact.github.io/ecosysteme-alpha-fle/`.

Aucune page pédagogique n'a été supprimée. Aucun dossier `assets/`, `pages/` ou contenu éditorial n'a été vidé.
